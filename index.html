Target acquisition experience in VR FPS game

In this project, I am going to compare three different input devices in FPS, a first-person shooting game. Using Virtenvironment , I developed a simple FPS game with unity engine which contains first character movement control. I used 3 different input devices, Razer Hydra, a gamepad, and a joystick with three different gaming levels. The first includes targets that are large, the second medium, and the last, and the hardest, small. Each participant must hit at least 50 targets per device. When they destroy 50 targets, they should start working with second and then the third device. The results of the experiment will be analyzed to find which input device has better performance for the player. In addition, we are going to know what is user experience from these kinds of game (FPS game in VR). We will give them questionnaires about their feelings toward the game, for example what they found challenging, positive, and negative.
